Bambu Studio 使用方法マニュアル

1. インストール

まずはブラウザで「Bambu Studio」と検索してください．そうするとこのページが出てくると思いますので，これをクリックしてください．

ページをクリックすると，下の画面が出てくると思うので，お使いのパソコンに合った方をダウンロードしてください．

ダウンロードが完了するとエクスプローラーやFinderに保存されていると思うので，これをご自身の好きなフォルダーに保存しておいてください．(任意)

ダウンロード完了した「Bambu_Studio_win_public」をクリックして起動させてください．

クリックした際に「このアプリがデバイスに変更を加えることを許可しますか」と聞かれることがありますが，「はい」で大丈夫です．

ダウンロードした「Bambu Studio」を開くと初期設定が始まります．

初めの画面は「Bambu Studio 02.00.03.54 セットアップへようこそ．セットアップでは，Bambu Studio 02.00.03.54 のインストールを順を追って案内します．インストールを開始する前に，他のすべてのアプリケーションを終了させることをおすすめします．これにより，コンピューターを再起動せずに必要なシステムファイルを更新できるようになります．「次へ」をクリックして続行してください．」
と書いてありますので，「Next ＞」をクリックしてください．

次の画面では「Bambu Studio 02.00.03.54 をインストールする前に，ライセンス条項を確認してください．キーボードの「Page Down」キーを押すと，契約書の続きを見ることができます．契約内容に同意する場合は，「同意する（I Agree）」をクリックして続行してください．この契約に同意しないと，Bambu Studio 02.00.03.54 をインストールすることはできません．」
と書いてありますので，「I Agree」をクリックしてください．

次の画面では「Bambu Studio 02.00.03.54 をインストールするフォルダーを選択してください．セットアップは，次のフォルダーに Bambu Studio 02.00.03.54 をインストールします．別のフォルダーにインストールしたい場合は，「参照（Browse）」をクリックして，別のフォルダーを選んでください．「インストール（Install）」をクリックすると，インストールが開始されます．」
と書いてありますので，必要に応じてフォルダーを参照し，「Install」をクリックしてください．(画像を撮ることを忘れていたのですみませんがここの画像はありません．)
インストールが完了すると下のような画像の画面になるので，「Next ＞」クリックしてください．

次の画面では「インストール成功．Bambu Studio を起動する．以下のファイル形式を Bambu Studio に関連付ける：.3mf ファイルを Bambu Studio に関連付ける：.stl ファイルを Bambu Studio に関連付ける：.step / .stp ファイルを Bambu Studio に関連付ける」
と書いてありますので，すべてにチェックを入れたまま「Close」をクリックしてください．

もしかすると「Close」をクリックした後に下の画像のような警告が出るかもしれません．内容は「コマンドライン引数 0 を Unicode に変換できなかったため，無視されます」と書いてありますが，「OK」で大丈夫です．

2. 起動後セットアップ

起動が終わると下の画像の画面に切り替わります．「利用開始」をクリックして「地域選択・ユーザ体験向上プログラム」などのセットアップを済ませてください．

セットアップの途中で使用するプリンターの選択をする画面になりますが，今回使用するプリンターは「Bambu Lab A1 mini」なので最低限そこだけにチェックマークを入れて次へ進んでください．ただ，Bambu Lab社の製品だけでなく他社の製品も選択できるようになっているので，自分の持っているプリンターや他に使用することのあるプリンターも選択しておくといいと思います．

次にフィラメント選択ですが，これはデフォルトのまま特に変更はしなくて大丈夫です．あとから設定もできます．

Bambu ネットワークプラグインはデフォルトのチェックマークの入ったまま「完了」をクリックしてください．
ネットワークプラグインのインストール完了後に「パブリックネットワークとプライベートネットワークにこのアプリへのアクセスを許可しますか？」と聞かれることもありますが，「許可」で大丈夫です．

3. 新規登録・サインイン

ここからは新規登録・サインインを行っていきます．まず，「サインイン/登録」をクリックしてください．

クリックするとこのページに来るので，アカウントがある方はサインイン，アカウントがない方はアカウント作成をクリックしてください．今回はアカウント作成の手順を解説していきます．

アカウント登録をクリックすると下の画像のページに行くので，メールアドレス・認証コード・パスワードを入力してください．メールアドレスは学校のものでもいいですが，卒業後も使用する可能性があるのであれば，個人のメールアドレスを登録してください．(なぜかicloudのメールが使えなかったので注意してください)

登録が完了すると，先ほどのサインインページに行くので．先ほど登録したメールアドレスとパスワードを入力してサインインしてください．

4. 印刷準備

ここからは印刷方法について説明していきます．まず作成した3Dモデルを印刷可能なファイルに変換します．今回私はFusion360を使用したのでFusion360で説明していきます．
Fusion360で「ファイル」から「3Dプリント」をクリックしてください．

クリックするとこの画面が表示されるので，出力形式は「3mf」のままで印刷したいオブジェクトを選択して「OK」をクリックしてください．複数のオブジェクトをがある場合は，1ファイルにまとめるか，1ボディ1ファイルにするか選択してください．

「OK」をクリックすると以下の画面が出てきますので，Fusion360内の保存先の確認，ローカルコンピュータに保存(こちらも保存先を確認すること)，これらを確認して「保存」をクリックしてください．

ここまで出来たらBambu Studioに移動してください．
Bambu studioの「準備」をクリックしてください．

次に，先ほどエクスポートした3mfファイルを下記の四角で囲んでいる範囲内にドラック＆ペーストしてください．

読み込み時に下記のようなエラーが出ると思いますが，Fusion360でモデルを作成したためこのエラーは出て当然のものです．なので「OK」で大丈夫です．

1ファイルに複数のパーツがある場合には下記のような文言が出てきますがこれも「はい」で大丈夫です．（もし，作成したモデルの片方が浮遊しているなら「いいえ」でもいいですが，分からない場合はここで「はい」を選択してもあとから準備画面の上方のツールバーにある「オブジェクトに分割」から分割することができます．）

プレート上にモデルが表示されたら，先ずはプリンターを選択します．今回は「Bambu Lab A1 mini」を使用します．

次に，使用するフィラメントを選択します．今回は「Generic PLA」を使用します.

次に，「プロセス」の「詳細」をオンにし，「サポート」の「有効化」もオンにします．

次に，「プロセス」内の「強度」の「トップ面層数」をデフォルトの「5」から「3」にします．(これは別にしなくてもいいですが，トップの面はものによりますが今回はそんなに固くなくてもいいので「3」にしました．)

次に，左上の「プレビュー」をクリックして，右上の「スライス」をクリックしてください．そうするとサポート材が必要なところにはサポートが表示されます．

次に，右上の「造形開始」の左横の下向きのマークを押すと画像のような項目が出てくるので，その中から「エクスポート(全プレート)」をクリックしてください．

次に，プリンタにデータを送信するためにプリンタに刺さっているSDカード内にデータをエクスポートしていきます．まずはSDカードを差し込めるUSBを用意します．

次に，Bambu Lab A1miniからSDカードを抜きます．1回押し込むと出てきます．

抜き取ったSDカードを先ほど用意したUSBに差し込んでパソコンにUSB接続させます．

Bambu Studioに戻って，「エクスポート(全プレート)」(画像では「エクスポート」になっていますが「エクスポート(全プレート)」が正しいです．)をクリックし，SDカードの刺さっているUSB内に保存します．

5. 印刷

最後に印刷をしていきます．先ほどエクスポートしたSDカードをBambu Lab A1 miniに戻し，プリンタの背面にある電源をつけます．

電源をつけると下記の画面が出てきますので，矢印の指す画面を選択してください．

次に，今回プリントするモデルを選択してください．

選択すると下記の画面が出てきますので，右上の「Print」を選択してください．そうするとプリントが開始されます．

別に絶対しなければいけないことではないですが，プリント中にプリントの速さを選択することができます．まず，下記の画面の右端の囲っている部分をどこでもいいので選択します．

次に，下記の画面の矢印のところを選択します．(私はもうすでに速度を変更しているので166％になっていますが気にしないでください．)

選択すると速度の変更画面に移動するので一番下の最大パーセントを選択します．

印刷が終了しましたら，音が鳴りますので，音が鳴ってから最低でも2分は放置してからプリントしたものを台座から取ってください．印刷が終了してすぐにプリントしたものをさわると，やけどをする可能性やフィラメントがまだ完全に固まっていないと変形してしまう可能性があります．

トラブルシューティング

1. Bambu Studioのトップ画面が英語表記だった時

まず，左上の「ファイル」横の「Ｖ」をクリックし，その後に「設定(Settings)」をクリックしてください．

その次に，「言語(Langage)」から日本語を選択してください．

その次に，保存されていない内容があるといわれた場合は保存を行い，再起動を要求されるので，再起動をすれば完了です．

2. フィラメントの交換方法(PLAからPLA)

最初にフィラメントの外し方について解説していきます．まず，初めの画面の「filament」を選択します．

次に，今あるフィラメントを外すために「Unload」を選択します．

選択すると下記の画面になり，フィラメントを外し始めます．

途中で「フィラメントを抜いてください」という表示が出ますので，プリンターからフィラメントを引っ張りぬいてください．抜けたら「Retry」を選択してください．これでフィラメントの取り外し完了です．

次に，新しいフィラメントの取付方法について解説していきます．まずはフィラメントを外すとき同様初めの画面の「filament」を選択し，今回はフィラメントを取り付けるので「load」を選択します．

選択すると下記の画面になり，フィラメントを取り付ける準備をし始めます．

準備中にあらかじめフィラメントをプリンターに差し込んでこれ以上差し込めなくなるまでセットしておきます．

フィラメントを差し込んでから少し経つと以下のような表示が出てきます．これは「プリンター内に残っていたフィラメントを出しています，ノズルの先から新しいフィラメントが出ているか確認してください」という内容のものなのでノズルの先を確認して新しいフィラメントが出ていれば「Done」出ていなければ「Retry」を選択してください．フィラメントが出ていない場合はフィラメントがノズルの先から出てくるまで繰り返し「Retry」を選択してください．「Done」を押した後はプリンターがフィラメントの取付を完了させてくれるのでこれで作業は完了です．
